# dmspc-system-communication
Code for prototyping system communication for the ngRadar project. This code is written by the Data Management & Software Prototyping Cohort (DMSPC).

# Prerequisites - Are the below system utilities still necessary now that we have containers?
Before running the Django server, install these system-level tools:
- **macOS**: `brew install curl jq kcat`
- **Linux**: `sudo apt install curl jq kafkacat`
